# DealScope — Product Deals Aggregator

Professionalized repository for the DealScope Streamlit application. This project aggregates product listings from Flipkart and Croma and presents summarized best deals.

See the `docs/` folder for detailed documentation including installation, deployment, and architecture.

Quick start

1. Copy `.env.example` to `.env` and fill required variables.
2. Create and activate a virtualenv and install requirements:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

3. Run locally:

```bash
DISABLE_LLM=true ./.venv/bin/streamlit run main.py
```

For full documentation see `docs/INSTALLATION.md`, `docs/DEPLOYMENT.md` and `docs/OVERVIEW.md`.
